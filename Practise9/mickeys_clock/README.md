# Mickey's Clock Application
Программа отображает текущее системное время (минуты и секунды) с использованием рук Микки Мауса в качестве стрелок.

## Как запустить
1. Убедитесь, что установлена библиотека `pygame`.
2. Запустите файл `main.py`.

## Особенности
- Синхронизация с системным временем в реальном времени.
- Вращение графики через `pygame.transform.rotate`.
