Music Player with Keyboard Controller
Objective
Build an interactive music player with real-time keyboard controls and dynamic playlist management.

 Keyboard Controls
The player is fully controlled via the keyboard for a seamless experience:

[P] — Play: Start the current track from the beginning.

[S] — Stop: Completely stop the playback.

[N] — Next: Switch to the next track in the folder (circular playlist).

[B] — Back: Switch to the previous track.

[Q] — Quit: Close the application safely.
