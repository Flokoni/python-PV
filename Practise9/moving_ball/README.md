## 3.3 Moving Ball Game
Интерактивное приложение с управляемым объектом.

### Функционал:
* Отрисовка шарика радиусом 25 пикселей.
* Управление через стрелки (Up, Down, Left, Right).
* Система контроля коллизий: шарик не может выйти за пределы окна.
* Скорость перемещения: 20 пикселей за одно нажатие.

### Как запустить:

python moving_ball/main.py
